# Basic WebApp Projects

This repository contains three simple web applications: a basic counter website, a login webpage, and a palindrome checker. These projects are implemented using HTML, CSS, and JavaScript.

## Introduction

This repository serves as an educational resource for learning the basics of web development using HTML, CSS, and JavaScript. Each project demonstrates fundamental web development concepts and provides a starting point for further exploration.

## Projects

### Counter Website

A simple web page that allows the user to increment, decrement, and reset a counter.

### Login Webpage

A basic login form that captures user input for a username and password.

### Palindrome Checker

A web page that checks if a given word or phrase is a palindrome.

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

- A web browser (e.g., Chrome, Firefox, Edge)
- A code editor (e.g., VSCode, Sublime Text)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/P-JEYANTHAN/basic-webapp-projects.git



Counter Website
Open counter/index.html in your browser.
Use the buttons to increment, decrement, and reset the counter.

Login Webpage
Open login/index.html in your browser.
Enter a username and password, then click the "Login" button.

Palindrome Checker
Open palindrome/index.html in your browser.
Enter a word or phrase, then click the "Check" button to see if it is a palindrome.
